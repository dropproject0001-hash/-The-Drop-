## 2026-06-13 - [A11y and Keyboard Nav Improvements]
**Learning:** Core UI components (EpicModal, Toast, TacticalToast, LoadingSpinner, EmptyState) were missing ARIA attributes and proper roles. Tactical buttons and navbar controls lacked clear focus-visible indicators for keyboard users.
**Action:** Implemented ARIA labels, roles, and focus-visible states across key components to improve screen reader support and keyboard navigation.
